# 俄罗斯方块游戏 - 项目说明文档

## 📋 项目概述

本项目是一个基于原生HTML、CSS、JavaScript开发的俄罗斯方块网页游戏，已在原有基础上进行了全面升级和完善，实现了课程要求的所有功能。

### 技术栈
- **前端技术**：HTML5、CSS3、JavaScript (ES6+)
- **图形渲染**：Canvas 2D API
- **音频处理**：Web Audio API
- **数据存储**：localStorage
- **响应式设计**：CSS Grid、Flexbox、Media Queries

---

## 🎮 功能说明

### 一、游戏系统完善（必须功能）

#### 1.1 增强计分系统
**实现逻辑**：
- 消除行数计分规则：1行=100分，2行=300分，3行=600分，4行=1000分
- 最终得分 = 基础分数 × 等级系数
- 最高分自动保存到localStorage
- 游戏结束后自动更新最高分记录

**使用方法**：
1. 游戏过程中，分数实时显示在右侧面板
2. 每消除一行或多行，系统自动计算并累加分数
3. 最高分显示在分数下方
4. 刷新页面后，最高分数据不丢失

#### 1.2 难度选择系统
**实现逻辑**：
- 游戏开始前显示难度选择界面
- 提供三个难度档位：
  - **简单**：下落速度1000ms，网格10×20
  - **普通**：下落速度800ms，网格10×20
  - **困难**：下落速度500ms，网格12×24

**使用方法**：
1. 点击「开始游戏」按钮
2. 在弹出的难度选择界面中选择难度
3. 选择后自动进入对应难度的游戏

#### 1.3 游戏流程控制
**实现功能**：
- **开始游戏**：初始化游戏、生成方块、开始计时
- **暂停游戏**：暂停方块下落与分数更新，显示暂停界面
- **重新开始**：重置分数和游戏状态
- **游戏结束**：显示结算界面（本次分数、最高分、操作按钮）

**使用方法**：
- 点击相应按钮控制游戏
- 按 P 键暂停/继续游戏
- 游戏结束后可选择「再来一局」或「回放」

### 二、视觉与动效

#### 2.1 方块视觉效果
**实现特点**：
- 方块使用渐变色效果，增加立体感

**实现主题**：

### 三、进阶功能

#### 3.1 本地排行榜
**实现功能**：
- 记录多次游戏成绩（分数、消除行数、时间戳）
- 按分数降序排列展示
- 最多保存100条记录
- 排行榜数据保存在localStorage

**使用方法**：
1. 游戏结束后自动保存成绩
2. 点击「🏆 排行榜」按钮查看历史记录
3. 点击「查看排行榜」按钮从结算界面查看

**数据格式**：
```javascript
{
    score: 1000,      // 分数
    lines: 50,       // 消除行数
    timestamp: 1234567890  // 时间戳
}
```

#### 3.2 操作回放功能
**实现功能**：
- 游戏中记录所有操作（方向键、旋转、下落）
- 游戏结束后可点击「回放」按钮复现过程
- 回放进度条实时显示
- 支持暂停、调整速度、停止

**使用方法**：
1. 游戏结束后点击「回放」按钮
2. 使用控制按钮调整回放：
   - 暂停/继续
   - 调整速度（1x, 2x, 3x, 0.5x）
   - 停止回放
3. 回放完成后自动返回主界面

**代码位置**：
- `recordOperation()` - 记录操作
- `startReplay()` - 开始回放
- `replayNextOperation()` - 回放逻辑

#### 3.3 触屏手势支持
**实现功能**：
- 左滑：向左移动方块
- 右滑：向右移动方块
- 上滑：旋转方块
- 下滑：加速下落
- 最小滑动距离：30px

**使用方法**：
1. 在游戏画布上进行触摸滑动
2. 根据滑动方向执行对应操作
3. 触屏操作与键盘操作逻辑一致

**代码位置**：
```javascript
canvas.addEventListener('touchstart', handleTouchStart);
canvas.addEventListener('touchend', handleTouchEnd);
```

## 📝 注意事项

1. **首次加载**：音效需要用户交互后才能播放
2. **localStorage**：数据存储在浏览器本地，清理浏览器会清除数据
3. **性能**：Canvas游戏性能良好，但在低性能设备上可能出现卡顿
4. **触屏设备**：建议在触屏设备上使用触屏操作体验更佳
